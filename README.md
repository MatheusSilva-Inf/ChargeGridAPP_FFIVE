# ChargeGridAPP_FFIVE
O ChargeGrid APP é uma aplicação de simulação desenvolvida em Python pela equipe FFIVE. Esse é um projeto acadêmico em busca de criar uma solução comercial para a empresa GoodWe. O aplicativo visa permitir para donos de eletropostos recargas de carros elétricos de forma rápida, intuitiva e informativa.
